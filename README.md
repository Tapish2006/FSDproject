# HealthLink

A React-based doctor appointment booking app built with Vite and Tailwind CSS. Users can browse doctors, view details, and request appointments using a clean, responsive interface.

## Features

- Browse doctors by specialty and services
- View detailed doctor profiles
- Book appointments with a booking form
- User login and signup pages
- Responsive layout for desktop and mobile
- Client-side routing with React Router DOM

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router DOM
- ESLint

## Installation

1. Clone the repository

```bash
git clone https://github.com/Kunalchoudhary2977/HealthLink-Doctor-appointment-app-.git
cd HealthLink-Doctor-appointment-app-
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open the app in your browser

```text
http://localhost:5173
```

## Available Scripts

- `npm run dev` - start local development server
- `npm run build` - build the app for production
- `npm run preview` - preview the production build locally
- `npm run lint` - run ESLint checks

## Project Structure

- `src/App.jsx` - main application component
- `src/components/` - reusable UI components
- `src/pages/` - page views like Home, Doctors, Dashboard, Login, Signup
- `src/data/doctors.jsx` - sample doctor and appointment data
- `src/reducer.jsx` - app state management logic
